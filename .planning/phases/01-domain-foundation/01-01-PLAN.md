   ```kotlin
   package com.acalc.domain

   enum class LengthUnit(val displayName: String) {
       MM("mm"), CM("cm"), M("m"), KM("km"),
       INCH("in"), FOOT("ft"), YARD("yd"), MILE("mi")
   }

   enum class WeightUnit(val displayName: String) {
       MG("mg"), G("g"), KG("kg"),
       OZ("oz"), LB("lb"), TON("t")
   }

   enum class VolumeUnit(val displayName: String) {
       ML("mL"), L("L"),
       TSP("tsp"), TBSP("tbsp"), CUP("cup"), FL_OZ("fl oz"), GALLON("gal")
   }

   enum class TempUnit(val displayName: String) {
       CELSIUS("°C"), FAHRENHEIT("°F"), KELVIN("K")
   }

   enum class AreaUnit(val displayName: String) {
       SQ_MM("mm²"), SQ_CM("cm²"), SQ_M("m²"), SQ_KM("km²"),
       SQ_IN("in²"), SQ_FT("ft²"), ACRE("ac")
   }

   enum class SpeedUnit(val displayName: String) {
       M_PER_S("m/s"), KM_PER_H("km/h"), MPH("mph"), KNOTS("kn")
   }
   ```

   Unit counts by category (matching REQUIREMENTS.md):
   - Length (CONV-04): 8 units — mm, cm, m, km, inches, feet, yards, miles
   - Weight (CONV-05): 6 units — mg, g, kg, oz, lb, ton (metric tonne per research recommendation)
   - Volume (CONV-06): 7 units — ml, L, tsp, tbsp, cup, fl oz, gallon (all US customary)
   - Temperature (CONV-07): 3 units — Celsius, Fahrenheit, Kelvin
   - Area (CONV-08): 7 units — sq mm, sq cm, sq m, sq km, sq in, sq ft, acres
   - Speed (CONV-09): 4 units — m/s, km/h, mph, knots
